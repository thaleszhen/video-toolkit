# Video Toolkit 可卖 MVP 清单

## 目标客户

- 播客和访谈团队：需要把一条长视频快速转成 YouTube、短视频和课程平台版本
- 小型 MCN / 内容工作室：需要批量做平台适配、压缩、水印和音频标准化
- 知识付费与企业培训团队：需要把成片批量导出为多平台交付版本

## 核心卖点

- 本地执行，素材不必上传第三方平台
- 基于 JSON 工作流，便于复制、复用、沉淀 SOP
- 能批量处理现成视频，适合团队交付而不是单次手工剪辑

## 可卖 MVP 必须具备

### 1. 真的能交付

- 支持内置预设和自定义工作流文件
- 支持目录级批量处理
- 输出文件命名稳定，可预测
- 出错后能看到清晰的失败列表

### 2. 真的能复用

- `create-workflow` 能基于预设生成可编辑模板
- 工作流可以直接通过 `--config` 执行
- 文档里给出面向交付场景的命令示例

### 3. 真的可靠

- 核心参数校验不能把 `0` 误判为缺失
- 裁剪定位逻辑必须正确
- CLI 帮助和关键路径要有自动化测试覆盖

## 当前版本已补齐

- 自定义工作流执行：`workflow --config <file>`
- 批量处理：`workflow --input-dir <dir> --output-dir <dir>`
- 批量 dry-run 预览
- `create-workflow --from-preset <name>`
- 裁剪定位修复
- 工作流校验修复

## 第一阶段商业化包装

- 交付形态：CLI + 配套 SOP + 预设模板
- 收费方式：按团队收“部署/配置费 + 月服务费”
- 建议报价：
  - 一次性配置费：`1999-4999 RMB`
  - 月服务费：`999-2999 RMB`

## 推荐销售话术

- “把已经剪好的主视频，一键导出成 YouTube / 短视频 / 培训版本。”
- “素材本地处理，不需要上传到第三方 SaaS。”
- “工作流可沉淀，团队成员可以复用同一套交付标准。”

## 接下来最该做的三件事

1. 增加批量任务报告和失败重试
2. 增加面向交付的预设：课程版、播客版、品牌版
3. 做一个轻量本地 Web UI，降低非技术用户门槛
